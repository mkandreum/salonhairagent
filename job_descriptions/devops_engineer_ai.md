# DevOps Engineer (AI/ML Infrastructure)

## About Us
We are an AI-powered process automation company dedicated to transforming business operations through intelligent agent systems. Our mission is to build AI agents that automate complex workflows, reduce manual effort, and drive operational excellence.

## Position Overview
We are seeking a DevOps Engineer with specialized experience in AI/ML infrastructure to join our founding engineering team. You will be responsible for building and maintaining the infrastructure that powers our AI agent systems, ensuring reliability, scalability, and cost-effectiveness of our LLM-based services.

## Key Responsibilities
- Design and implement CI/CD pipelines for AI model deployment and agent workflow updates
- Set up and manage cloud infrastructure (AWS/Azure/GCP) for AI/ML workloads
- Implement containerization strategies using Docker and Kubernetes for LLM applications
- Establish monitoring, logging, and observability for AI systems (MLflow, Weights & Biases, Prometheus, etc.)
- Manage infrastructure as code using Terraform or CloudFormation
- Optimize costs for LLM API usage and GPU-accelerated infrastructure
- Ensure security and compliance for AI systems (data privacy, model governance, access controls)
- Implement backup, disaster recovery, and high availability strategies for AI services
- Collaborate with ML engineers to streamline model deployment and experimentation workflows
- Troubleshoot and resolve infrastructure issues affecting AI agent performance

## Required Qualifications
- 3+ years of professional DevOps or Site Reliability Engineering experience
- Strong experience with cloud platforms (AWS, Azure, or GCP) including AI/ML services
- Proficiency with containerization technologies (Docker, Kubernetes)
- Experience with CI/CD tools (Jenkins, GitLab CI, GitHub Actions, etc.)
- Infrastructure as Code expertise (Terraform, CloudFormation, or similar)
- Knowledge of monitoring and logging stacks (ELK, Prometheus/Grafana, Datadog, etc.)
- Experience with Linux system administration and networking
- Proficiency with scripting languages (Python, Bash, etc.)
- Understanding of microservices architecture and API management
- Familiarity with version control systems (Git) and collaborative development practices

## Preferred Qualifications
- Experience with GPU-accelerated computing and CUDA
- Knowledge of MLOps platforms and model serving solutions (Seldon, KFServing, etc.)
- Familiarity with vector databases and their operational requirements
- Experience with service meshes (Istio, Linkerd) for microservices communication
- Background in AI/ML workload optimization and performance tuning
- Understanding of data engineering concepts and ETL pipelines
- Experience with security frameworks for AI/ML systems
- Knowledge of cost optimization strategies for cloud AI workloads
