# Senior Software Engineer (Full Stack - AI Focus)

## About Us
We are an AI-powered process automation company dedicated to transforming business operations through intelligent agent systems. Our mission is to build AI agents that automate complex workflows, reduce manual effort, and drive operational excellence.

## Position Overview
We are seeking a Senior Software Engineer with full-stack expertise to join our founding engineering team. You will work closely with the CTO to build our core AI automation platform, implementing features that leverage LangGraph, LLMs, and modern web technologies to create intelligent agent systems for business process automation.

## Key Responsibilities
- Design and implement core automation features using LangGraph and AI agent frameworks
- Build and maintain scalable API services for agent communication and orchestration
- Develop React-based user interfaces for agent configuration and monitoring
- Collaborate on system architecture for scalable AI agent workflows
- Implement integrations with external systems, databases, and third-party APIs
- Mentor junior engineers and establish engineering best practices
- Participate in code reviews, architectural decisions, and technical planning
- Stay current with emerging AI/ML technologies and recommend adoption strategies

## Required Qualifications
- 5+ years of professional software development experience
- 3+ years of experience with TypeScript/JavaScript
- 3+ years of experience with React and Node.js
- Strong foundation in API design (REST, GraphQL, WebSocket)
- Experience with AI agent frameworks (LangGraph, LangChain, LlamaIndex, etc.)
- Hands-on experience with LLM integration (OpenAI GPT, Anthropic Claude, etc.)
- Understanding of vector databases and embedding technologies
- Experience with cloud platforms (AWS/Azure/GCP) and containerization (Docker)
- Proficiency with Git and modern DevOps practices
- Excellent problem-solving and communication skills

## Preferred Qualifications
- Experience with RAG (Retrieval-Augmented Generation) systems
- Knowledge of MLOps and MLflow for model lifecycle management
- Familiarity with monitoring and observability tools for AI systems
- Background in process automation or workflow management systems
- Open-source contributions to AI/ML or agent frameworks
- Experience with testing frameworks for full-stack applications
- Understanding of microservices architecture and event-driven systems

## What We Offer

- Equity participation as an early founding team member
- Comprehensive benefits package including health insurance

- Opportunity to shape the technical direction of an AI startup
